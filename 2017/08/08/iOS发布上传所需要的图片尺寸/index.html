iOS发布上传所需要的图片尺寸

1. 桌面图标（App icon）

iPhone X/8 plus/7 plus/6 plus(@3x) : 180x180
iPhone 8/7/6s/6/5s/5/4s/4 (@2x) : 120x120

2.系统搜索框图标 （Spotlight search result icon）

iPhone X/8 plus/7 plus/6 plus(@3x) : 120x120
iPhone 8/7/6s/6/5s/5/4s/4 (@2x) : 80x80

3.系统设置图标 (Setting icon)

iPhone X/8 plus/7 plus/6 plus(@3x) : 87x87
iPhone 8/7/6s/6/5s/5/4s/4 (@2x) : 58x58

4.通知栏图标 （Notification）

iPhone X/8 plus/7 plus/6 plus(@3x) : 60x60
iPhone 8/7/6s/6/5s/5/4s/4 (@2x) : 40x40

5.App Store

1024x1024

6.启动页（Launch image）

下面仅仅指Portrait模式下：
1242 x 2688 XS MAX
828 x 1792 XR
1125 x 2436 X iOS11+
1242 x 2208 5.5 iOS8+
750 × 1334 4.7 iOS8+
640 × 960 2x iOS7+、iOS5、6
640 × 1136 Retina 4 iOS7+、iOS5、6

7.上传到App Store所需的各分辨率图

比较特殊的是iPhone X只有自己专属尺寸的图，而其它尺寸的图都可以用5.5寸图去代替，但是这个5.5寸图一定要注意尺寸大小是1242 2208，
同样iPad的图最大尺寸也是可以向下代替的。
| 3.5 | 640 x 960 |
| 4 | 640 x 1136 |
| 4.7 | 750 x 1334 |
| 5.5 | 1242 x 2208 |
| 5.8 | 1125 x 2436 |
| 9.7 | 2048 x 1536 |
| 12.9 | 2732 2048 |
